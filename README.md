# Pokeyo

宝可梦风格的网页小游戏：纯 HTML / CSS / JavaScript，无任何依赖，无需构建。

## 怎么玩

直接用浏览器打开 `index.html` 即可（双击就行）。

- **方向键 / WASD** 在地图上移动
- 走进 **深草丛** 有概率遭遇野生宝可梦
- 战斗中可以 **战斗 / 用背包道具 / 换宝可梦 / 逃跑**
- 用 **精灵球** 捕捉野生宝可梦（HP 越低越容易抓到）
- 走到 **✚ 治疗台** 恢复全队并自动存档（存档在浏览器 localStorage）
- 地图上的 **◓** 可以捡到精灵球
- 升级会学新招式，御三家 16 级会进化

## 玩法机制

- 8 种属性（火 / 水 / 草 / 电 / 岩石 / 飞行 / 虫 / 普通）与克制表，本系加成 1.5 倍
- 类初代的伤害公式、速度决定出手顺序
- 经验值按 level³ 曲线升级

## 文件结构

```
index.html      页面骨架
style.css       全部样式
js/data.js      属性表、招式、图鉴、地图、遇敌表
js/monster.js   宝可梦实例、伤害/经验计算
js/ui.js        侧边栏、提示等通用 UI
js/overworld.js 大地图渲染、移动、遇敌
js/battle.js    回合制战斗状态机
js/main.js      全局状态、存档、入口
```
